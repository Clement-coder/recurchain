# RecurChain Frontend

## Project Overview

RecurChain is a decentralized application (dApp) that allows users to create and manage autonomous recurring payment agents. It provides a comprehensive dashboard to track payments, manage wallet balances, view transaction history, receive notifications, and configure personal settings.

## Features

-   **Agent Management:** Create, edit, pause, resume, and delete recurring payment agents.
-   **Dashboard:** Overview of active agents, recent activities, and key metrics.
-   **Wallet:** Manage USDC balance, deposit, and withdraw funds.
-   **Transaction History:** View a detailed history of all recurring payments and transactions.
-   **Notifications:** Receive alerts for payment successes, failures, and low balances.
-   **Settings:** Configure profile information, bank accounts, notification preferences, security, and privacy settings.
-   **Responsive Design:** Optimized for various screen sizes, including mobile with a collapsible sidebar.

## Technologies Used

-   **Next.js 16 (App Router):** React framework for building full-stack web applications.
-   **React 19:** JavaScript library for building user interfaces.
-   **TypeScript:** Strongly typed superset of JavaScript.
-   **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
-   **Framer Motion:** A production-ready motion library for React.
-   **Lucide React:** A collection of beautiful and customizable SVG icons.

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/RecurChain.git
    cd RecurChain/frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. The page auto-updates as you edit the files.

## Usage

Once the application is running, you can:

-   Navigate to the `/dashboard` to see an overview of your agents.
-   Go to `/agents` to create and manage your recurring payment agents.
-   Visit `/wallet` to manage your funds.
-   Check `/history` for all your transaction records.
-   Configure your preferences in `/settings`.
